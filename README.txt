MULTI-TOURNAMENT WORKFLOW
───────────────────────────
  1. Run Tournament A → enter results
  2. Admin → Settings → "Archive & Start New"
     (or Admin → All Tournaments → Archive & New button)
  3. Tournament A is saved permanently
  4. Portal resets for Tournament B with same venue/org settings
  5. Repeat for as many tournaments as needed
  6. View any past tournament: Admin → All Tournaments
  7. Download PDF of any past tournament at any time
  8. Restore a past tournament to active if needed
